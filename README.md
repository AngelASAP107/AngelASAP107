## Proyectos destacados

**[Credit Risk — Home Credit](https://github.com/AngelASAP107/portfolio-credit-risk)**
Predicción de incumplimiento crediticio sobre 57M filas relacionales. Feature engineering desde 6 tablas secundarias (79 features nuevas), LightGBM con manejo de desbalance, ROC-AUC 0.7765, detección del 68.8% de incumplimientos reales. API REST en FastAPI + Docker.

**[International Football Analytics](https://github.com/AngelASAP107/football-analytics)**
Análisis y predicción sobre 153 años de fútbol internacional (49,000+ partidos). Random Forest con evaluación cronológica y feature engineering sin data leakage.
